**JPA:**

Java Persistence API: 数据持久化，对象/关系映射

类必须用javax.persistence.Entity注解

关系：

-   一对一：@OneToOne
-   一对多：@OneToMany
-   多对一：@ManyToOne
-   多对多：@ManyToMany





Spring Data JPA 常用接口

-   CrudRepository：增删改查
-   PagingAndSortingRepository：分页、排序
